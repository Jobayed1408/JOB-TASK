# Front-End Design Challenge: All-Inclusive Creative Services

A high-fidelity landing page section built from a Figma design. This project focuses on design accuracy, responsive layouts, and clean, maintainable vanilla code.

## 🚀 Live Demo
**https://job-task-project-jobayed.netlify.app/**

## 📋 Project Overview
This task involved translating a complex Figma UI into a functional, responsive web component using only core web technologies. The layout features a client logo showcase and a detailed service selection section with custom-styled interactive elements.

### Key Features:
* **Design Precision:** Matched typography, spacing, and color palettes exactly as specified in the Figma file.
* **Custom UI Components:** Built custom radio-style selection cards and badges without external UI libraries.
* **Fully Responsive:** Implemented a mobile-first approach using CSS Media Queries to ensure a seamless experience across all device sizes.
* **Vanilla Implementation:** No frameworks, pre-processors, or AI-generated code—strictly HTML5, CSS3, and JavaScript.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for better accessibility and SEO.
* **CSS3:** Flexbox for layout, CSS Variables for theme management, and transitions for interactivity.
* **JavaScript:** Minimal vanilla JS for interactive state management.
* **Font Awesome:** Integrated via CDN for high-quality vector iconography.
